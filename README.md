# Downwork – Freelance Platform Landing Page (Practice Project)

## 📌 Overview
This project is a front-end practice build inspired by the Upwork homepage.  
It was recreated to strengthen my understanding of HTML5 and CSS3 layout fundamentals.

The main focus of this project is building a structured, and clean layout without relying on responsiveness or advanced JavaScript.

---

## 🎯 Learning Objectives
- Build a complete multi-section landing page from scratch
- Practice semantic HTML5 structure
- Improve layout skills using Flexbox & Grid
- Develop a strong understanding of spacing, alignment, and component structure
- Organize assets and project structure professionally

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (basic interactions)

---

## ⚠️ Project Scope (Important)
This project is intentionally non-responsive.

It focuses only on:
- Desktop layout
- Structure and design implementation

Future improvements will include:
- Responsive design (mobile & tablet)
- Improved JavaScript using class-based toggling
- Enhanced accessibility and performance

---

## 📂 Features
- Multi-section landing page
- Navigation bar with dropdown concept
- Hero section with toggle interaction (Hire / Work)
- Service category grid
- Pricing cards
- Testimonials section
- Footer with structured links

---

## 🚀 Future Improvements
- Make the layout fully responsive using media queries
- Refactor JavaScript to follow best practices (class toggling)
- Improve accessibility (alt tags, semantic structure)
- Add animations and micro-interactions
- Optimize CSS using variables and reusable components

---

## 📖 Note
This project is inspired by the Upwork website and was built purely for educational and practice purposes.  
All branding has been modified, and the goal is to improve front-end development skills through real-world UI replication.
